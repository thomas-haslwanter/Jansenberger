Help

Installation

  • Install WinPython or Anaconda
  • Make sure your system environment is set correctly
    • Run "setup.bat" to ensure that the required environmental variables are set.
  • Install DB Browser for SQLite, in order to inspect and modify the database.

Use

Use "Settings|Defaults" to adjust the default settings.
